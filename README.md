# sc2
revamped UI for my previous project [sc](https://github.com/dekztah/sc)
soundcloud user stream built with AngularJS using Web Audio API

- dark, light, transparent mode
- 3 different background presets
- spectrum analyser
- oscilloscope
- playlist support
- toggle reposts
